## 需求分析
1. 支持往不同的地方输出日志
  - 终端
  - 文件
  - kafka
2. 日志分级别
  - Debug
  - Trace
  - Info
  - Warning
  - Error
  - Fatal
3. 支持开关
4. 完整日志信息: 时间、行号、文件名、日志级别、日志信息
5. 日志文件要切割
6.
